# DiagonalReplace — пошаговая диаграмма алгоритма

## Общая схема алгоритма

```mermaid
flowchart TD
    A([Начало]) --> B["Задаём n = 4"]
    B --> C["Создаём матрицу int[n][n]\nвсе элементы = 0"]
    C --> D["Печатаем исходную матрицу"]
    D --> E["Цикл: i = 0 .. n-1\nmatrix[i][i] = 10"]
    E --> F["Печатаем изменённую матрицу"]
    F --> G([Конец])
```

## Детальная работа цикла замены диагонали

```mermaid
flowchart TD
    START([Начало цикла]) --> INIT["i = 0"]
    INIT --> CHECK{"i < n (4)?"}
    CHECK -- Да --> SET["matrix[i][i] = 10"]
    SET --> INC["i++"]
    INC --> CHECK
    CHECK -- Нет --> END([Конец цикла])
```

## Пошаговое изменение матрицы

```mermaid
flowchart LR
    subgraph "Шаг 0: Исходная матрица"
        M0["0 0 0 0\n0 0 0 0\n0 0 0 0\n0 0 0 0"]
    end

    subgraph "Шаг 1: i=0 → matrix[0][0]=10"
        M1["⟨10⟩ 0  0  0\n 0  0  0  0\n 0  0  0  0\n 0  0  0  0"]
    end

    subgraph "Шаг 2: i=1 → matrix[1][1]=10"
        M2["10  0  0  0\n 0 ⟨10⟩ 0  0\n 0  0  0  0\n 0  0  0  0"]
    end

    subgraph "Шаг 3: i=2 → matrix[2][2]=10"
        M3["10  0  0  0\n 0  10  0  0\n 0  0 ⟨10⟩ 0\n 0  0  0  0"]
    end

    subgraph "Шаг 4: i=3 → matrix[3][3]=10"
        M4["10  0  0  0\n 0  10  0  0\n 0  0  10  0\n 0  0  0 ⟨10⟩"]
    end

    M0 --> M1 --> M2 --> M3 --> M4
```

## Связь индексов с позициями в матрице

```mermaid
flowchart TD
    subgraph "Главная диагональ: row == col"
        A["[0][0]"] --> B["[1][1]"] --> C["[2][2]"] --> D["[3][3]"]
    end

    subgraph "Матрица 4×4"
        direction TB
        R0["[0][0]  [0][1]  [0][2]  [0][3]"]
        R1["[1][0]  [1][1]  [1][2]  [1][3]"]
        R2["[2][0]  [2][1]  [2][2]  [2][3]"]
        R3["[3][0]  [3][1]  [3][2]  [3][3]"]
    end
```

> **Ключевая идея:** на главной диагонали индекс строки всегда равен индексу столбца (`i == j`), поэтому достаточно одного цикла с `matrix[i][i] = 10`.
